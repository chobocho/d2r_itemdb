# D2R 아이템 도감 v2.0

Diablo II: Resurrected 한국어 아이템/룬/룬워드 검색 데이터베이스

## 기술 스택

- 바닐라 JavaScript (프레임워크 없음)
- IndexedDB (브라우저 로컬 DB)
- HTML/CSS (반응형)

## 주요 기능

- 룬, 룬워드, 유니크, 세트, 퀘스트, 용병, 지역, 이벤트, 악마술사 10개 탭 검색
- 다중 키워드 검색 (AND 조건), 키워드 하이라이트
- 버전 기반 자동 데이터 업데이트 (IndexedDB)
- 사용자 커스텀 데이터 추가/유지
- 반응형 UI, 스크롤 탑, 이름순 정렬

## 파일 구조

```
index.html       - UI 및 CSS
app.js           - 메인 애플리케이션 로직 (바닐라 JS)
app.ts           - TypeScript 원본 소스 (미사용, 레거시)
data.json        - 아이템 데이터베이스
db_version.json  - 데이터 버전 관리
```
